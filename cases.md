# Playwright — тренировка локаторов  
**Сайт:** https://uitestingplayground.com/

Цель набора — прокачать навык **поиска устойчивых локаторов** в Playwright:  
semantic, relative, scoped, text, regex, aria, shadow DOM и т.д.

---

## 1. Dynamic ID — базовый
**Страница:** Dynamic ID  

**Задание:**  
Найти и кликнуть кнопку `Button`.

**Ограничения:**
- нельзя использовать `id`
- нельзя использовать позиционные XPath
- локатор должен работать при каждом перезапуске страницы

**Тренируется:** устойчивость локаторов без `id`
